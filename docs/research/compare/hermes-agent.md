# Hermes Agent

**Description:** A self-improving AI agent built by Nous Research.
**Key Features:**
- **Closed Learning Loop:** Creates skills from experience, improves them during use, and nudges itself to persist knowledge.
- **Memory System:** FTS5 session search with LLM summarization for cross-session recall. Compatible with Honcho dialectic user modeling.
- **Serverless Persistence:** Runs on Modal/Daytona, hibernating when idle to cut costs.
- **Multi-platform:** Lives in Telegram, Discord, Slack, etc., with a full TUI backend.
