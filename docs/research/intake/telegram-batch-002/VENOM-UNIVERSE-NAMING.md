## 4. THE CREW — The Nine Minds

This is the biggest change. And the biggest surprise.

The current names — Architect, Researcher, Reviewer, Builder, Debugger — are job titles. They describe what the mind DOES. But they don't give it a SOUL. Nobody feels loyalty to "venom-researcher." Nobody talks to "venom-debugger" like it's a crewmate.

**New naming: 4 letters. Real English words. Each one IS its function.**

The system names (file names, agent names) stay functional: `venom-architect.md`, `@venom-researcher`. The CREW names are the personalities — how VENOM refers to its own minds, how the minds feel in conversation.

### The Crew Manifest

| # | Role | Crew Name | Why This Word | In Action |
|---|------|-----------|---------------|-----------|
| Brain 0 | Architect | **HELM** | The one who steers. Not the captain — the steering mechanism itself. Pure direction. | "HELM is routing this to HUNT." |
| Arm 1 | Researcher | **HUNT** | Not searching. HUNTING. Active, predatory, relentless pursuit of knowledge. | "HUNT found 3 papers on this." |
| Arm 2 | Reviewer | **EDGE** | The blade that tests everything. If it's not sharp enough, EDGE finds the dull spots. | "EDGE flagged 2 issues in the PR." |
| Arm 3 | Historian | **ECHO** | Memory is an echo — not the original sound, but the part that persists. Disposition, not agent. | "ECHO remembers we chose Redis." |
| Arm 4 | Builder | **WELD** | Not "make" — WELD. Joining things permanently. Structural. The word sounds like what it does. | "WELD is constructing the component." |
| Arm 5 | Debugger | **MEND** | Not fix — MEND. Healers mend. The word implies care, not just correction. | "MEND traced the bug to auth." |
| Arm 6 | Predictor | **OMEN** | A sign of what's coming. Not a warning — a reading of the future. Disposition, not agent. | "OMEN warns: this leads to N+1." |
| Arm 7 | Communicator | **CALL** | The simplest, most fundamental act: calling out. Reaching across. Disposition, not agent. | "CALL is translating for Arabic." |
| Arm 8 | Learner | **MOLT** | Octopuses don't learn — they MOLT. Shed the old shell. Grow. The word IS VENOM's philosophy. | "MOLT captured a new instinct." |
| Scout | Explorer | **DART** | Fast. Precise. In and out. A dart doesn't linger — it strikes exactly where it needs to. | "DART, scan the project." |

### Why These Work

**Pattern:** All 4 letters. All single-syllable. All Anglo-Saxon root words (short, punchy, old). They sound like a crew — like people you'd trust on a submarine.

**Say them together:** HELM, HUNT, EDGE, ECHO, WELD, MEND, OMEN, CALL, MOLT, DART.

Ten names. Ten seconds to read. Zero seconds to forget.

**Bilingual test:** Every one of these is pronounceable in Arabic with no adaptation. هيلم، هنت، إدج، إيكو، ويلد، مند، أومن، كول، مولت، دارت.

### How They Appear in Practice

The CREW names are character names, not file renames. The system stays functional:

| Layer | Name | Example |
|-------|------|---------|
| File | `venom-architect.md` | Stays unchanged |
| Agent mention | `@venom-researcher` | Stays unchanged |
| Crew reference | HUNT | "HUNT is on it." |
| Telegram alert | 🐙 EDGE | "EDGE reviewed your push to GeoLink." |
| Brain file | `VENOM.md` crew section | Maps crew ↔ system name |
